# Product Overview

## DMARC Lens

DMARC Lens is a tool for analyzing and visualizing DMARC (Domain-based Message Authentication, Reporting & Conformance) email security reports. The project aims to help organizations understand their email authentication posture and identify potential security issues.

## Key Features
- DMARC report ingestion
- DMARC report parsing and analysis
- Visualization of email authentication data
- Security insights and recommendations
- Domain reputation monitoring

## Target Users
- Email administrators
- Security teams
- IT professionals managing email infrastructure
